Still

Manual and scheduled focus with Screen Time shields, Still Mode, plus optional alarms you dismiss by walking or scanning your own QR code.

Purpose

Still is built for people who want to reduce distractions on their own terms: you choose which apps, categories, and websites are limited during focus. You can run a one-off timed session, set up weekly schedules (Still Pro), or use Still Mode with your own QR code. Alarms are optional and designed so you finish waking up with a short walk or a sticker you place away from your bed.

Focus & Still Mode

  • Manual focus — pick duration, apps/sites, and optional groups; shields apply for the session.
  • Still Mode — block chosen apps until you scan your printed QR and confirm to exit.
  • Cheat breaks — optional short breaks with a visible countdown (including Live Activity where supported), then shields return.

Scheduled blocks (Still Pro)

Recurring weekly windows can automatically start a focus-style session: same shield behavior and cheat flow as a manual session, started for you when the window begins.

  • Progress and controls for the active scheduled session appear under Scheduled Blocks on the Focus tab (the main Focus Session card points you there instead of duplicating the timer).
  • To limit battery use on long blocks, Still does not run a continuous Dynamic Island / Lock Screen timer for the scheduled session itself. Cheat breaks still use the same Live Activity countdown as manual focus and Still Mode.
  • While a scheduled session is running, you cannot start a separate manual focus session or Still Mode.
  • If you break out of a scheduled session early, that block’s schedule is turned off (same as disabling it when editing the block), so the recurring rule stops until you turn it back on.

Highlights

  • Screen Time integration: Family Controls, Managed Settings, and Device Activity for sessions and schedules
  • Optional alarms: system alarms on supported iOS versions, with walk-to-dismiss or QR dismiss
  • Data stays on your device for core focus features; see our privacy summary
